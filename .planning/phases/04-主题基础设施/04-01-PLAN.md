<task type="auto">
  <name>任务 1: 创建主题类型定义和 5 个预设主题配置</name>
  <files>src/core/theme.ts</files>
  <action>
创建 `src/core/theme.ts` 文件，定义主题类型系统和 5 个预设主题配置：

**类型定义：**
1. `ThemeId`：字面量联合类型 `'neon' | 'sky' | 'forest' | 'sunset' | 'sakura'`
2. `ThemeColors`：interface，包含以下属性（使用注释分组）：
   - 基础颜色：bgPrimary, bgSecondary, textPrimary, textSecondary, border, tileEmpty
   - 方块颜色：tileColors (Record&lt;number, string&gt;)
   - 特殊效果：gradientStart, gradientEnd
3. `ThemeConfig`：interface，包含 id (ThemeId), displayName (string), colors (ThemeColors)

**5 个主题配置：**
根据 CONTEXT.md 中的决策和 Tile.vue 的经典颜色作为基准，创建 THEMES 对象：

1. **霓虹暗色 (neon)**:
   - 基于 Tile.vue 的经典配色
   - 暗色背景 (#0f172a, #1e293b)
   - 霓虹渐变 (#0f172a → #581c87)
   - 方块颜色使用经典 2048 配色（从 Tile.vue 提取）

2. **天空蓝 (sky)**:
   - 蓝色系主题
   - 浅蓝背景 (#e0f2fe, #bae6fd)
   - 深蓝渐变 (#0284c7 → #0c4a6e)
   - 方块颜色使用蓝色系渐变

3. **森林绿 (forest)**:
   - 绿色系主题
   - 浅绿背景 (#dcfce7, #bbf7d0)
   - 深绿渐变 (#16a34a → #14532d)
   - 方块颜色使用绿色系渐变

4. **日落橙 (sunset)**:
   - 橙色系主题
   - 浅橙背景 (#ffedd5, #fed7aa)
   - 深橙渐变 (#ea580c → #7c2d12)
   - 方块颜色使用橙色系渐变

5. **樱花粉 (sakura)**:
   - 粉色系主题
   - 浅粉背景 (#fce7f3, #fbcfe8)
   - 深粉渐变 (#db2777 → #831843)
   - 方块颜色使用粉色系渐变

**关键规则：**
- 所有颜色值使用十六进制格式 (#RRGGBB)
- 方块颜色 tileColors 必须包含所有 2048 游戏数字：2, 4, 8, 16, 32, 64, 128, 256, 512, 1024, 2048
- 使用 `export const THEMES: Record<ThemeId, ThemeConfig>` 导出
- 添加 JSDoc 注释说明每个主题的特色

<task type="auto">
  <name>任务 2: 在 style.css 中定义 CSS 变量和主题选择器</name>
  <files>src/style.css</files>
  <action>
修改 `src/style.css` 文件，添加 CSS 变量定义和主题选择器：

**基础变量（默认值）：**
在 `:root` 中定义默认主题变量，使用 `--theme-*` 前缀：
- `--theme-bg-primary`: 主背景色
- `--theme-bg-secondary`: 次背景色
- `--theme-text-primary`: 主文字色
- `--theme-text-secondary`: 次文字色
- `--theme-border`: 边框色
- `--theme-tile-empty`: 空格子颜色
- `--theme-tile-2`, `--theme-tile-4`, ..., `--theme-tile-2048`: 方块颜色

**主题特定变量：**
为每个主题创建 `[data-theme='xxx']` 选择器，覆盖默认变量：
```css
[data-theme='neon'] {
  --theme-bg-primary: #0f172a;
  --theme-bg-secondary: #1e293b;
  /* ... 其他变量 */
}
```

**平滑过渡效果：**
添加全局过渡效果（符合 THEME-04 要求）：
```css
* {
  transition: background-color 0.2s ease, color 0.2s ease, border-color 0.2s ease;
}
```

**关键规则：**
- CSS 变量名必须与 theme.ts 中 ThemeColors 接口属性对应
- 变量名使用 kebab-case（如 `--theme-bg-primary`）
- 确保所有 5 个主题的变量都定义完整
- 过渡时间设置为 0.2s（在 0.15-0.3s 范围内）
- 保持现有的 `.dark` 类和 `.glass` 效果不变
