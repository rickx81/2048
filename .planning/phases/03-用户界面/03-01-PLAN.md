- 响应式设计：适配桌面和移动设备
- CSS Grid 布局：4x4 网格，方格之间有固定间距

<task type="auto">
  <name>任务 1：安装和配置 Tailwind CSS v4</name>
  <files>package.json, tailwind.config.js, src/style.css, src/main.ts</files>
  <action>
1. 安装 Tailwind CSS v4：
   ```bash
   npm install @tailwindcss/v4
   ```

2. 创建 Tailwind 配置文件 `tailwind.config.js`：
   - 启用暗色模式（class 策略）
   - 自定义颜色：定义数字方块的颜色光谱（2=青色，4=绿色，8=黄色，16=橙色...）
   - 自定义间距：定义网格间距
   - 配置主题扩展（theme.extend）

3. 更新 `src/style.css`（如果不存在则创建）：
   - 添加 Tailwind 基础样式：`@import "tailwindcss";`
   - 添加全局样式：body 背景、字体设置

4. 更新 `src/main.ts`（如果需要）：
   - 确保 CSS 导入正确

颜色方案定义（光谱渐变）：
- 2: 青色 (cyan-400/500)
- 4: 绿色 (green-400/500)
- 8: 黄色 (yellow-400/500)
- 16: 橙色 (orange-400/500)
- 32: 红色 (red-400/500)
- 64+: 紫色 (purple-400/500)

背景颜色（暗色/霓虹风格）：
- 主背景：深蓝/紫色渐变 (bg-gradient-to-br from-slate-900 via-purple-900 to-slate-900)
- 网格背景：半透明玻璃态 (bg-white/10 backdrop-blur-sm)
  </action>

  <name>任务 3：创建游戏容器组件（GameContainer.vue）</name>
  <files>src/components/GameContainer.vue</files>
  <action>
创建 `src/components/GameContainer.vue` 组件，提供响应式游戏布局容器：

**模板结构：**
```vue
&lt;template&gt;
  &lt;div class="game-container"&gt;
    &lt;!-- 头部区域（分数等）- 后续计划填充 --&gt;
    &lt;header class="game-header"&gt;
      &lt;!-- 占位符 --&gt;
    &lt;/header&gt;

    &lt;!-- 游戏网格区域 - 后续计划填充 --&gt;
    &lt;main class="game-board"&gt;
      &lt;!-- 占位符 --&gt;
    &lt;/main&gt;

    &lt;!-- 控制按钮区域 - 后续计划填充 --&gt;
    &lt;footer class="game-controls"&gt;
      &lt;!-- 占位符 --&gt;
    &lt;/footer&gt;
  &lt;/div&gt;
&lt;/template&gt;
```

**样式要求：**
- 使用 Tailwind CSS 类名
- 响应式设计：
  - 桌面端（>= 640px）：固定宽度容器（如 500px），居中显示
  - 移动端（< 640px）：全宽，适当内边距
- 暗色/霓虹风格背景（深蓝/紫色渐变）
- 玻璃态效果（backdrop-blur, bg-white/10）
- 内边距和圆角设计

**布局结构：**
- 最大宽度约束（max-w-md 或自定义）
- 居中对齐（mx-auto）
- 垂直间距（gap-4 或 gap-6）
- 内边距（p-4 或 p-6）

**容器尺寸策略（Claude's Discretion）：**
- 桌面端：固定容器约 500px 宽度，保持经典 2048 比例
- 移动端：自适应宽度，最小内边距 16px，确保在小屏幕上可玩
- 网格单元格：根据容器宽度动态计算，保持正方形比例

**脚本部分：**
```vue
&lt;script setup lang="ts"&gt;
import { useGameStore } from '@/stores/game'

const store = useGameStore()
// 可以在后续计划中添加更多逻辑
&lt;/script setup&gt;
```

**样式部分（scoped）：**
```vue
&lt;style scoped&gt;
/* 额外的自定义样式（如果有） */
.game-container {
  min-height: 100vh;
  display: flex;
  flex-direction: column;
}
&lt;/style scoped&gt;
```

注意：此任务只创建容器布局，不实现具体内容（头部、网格、控制按钮由后续计划实现）
