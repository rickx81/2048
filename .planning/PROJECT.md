# 2048 游戏

## 项目概述

基于 Vue 3 和 TypeScript 构建的经典 2048 网页游戏，专注于学习和实践 Vue 3 最佳实践。采用 TDD 开发模式和 Functional Core, Imperative Shell 架构模式。

## 核心价值

通过实际项目深入掌握 Vue 3 组合式 API、响应式系统、组件设计模式和现代前端工程化实践。

## 上下文

**项目背景：**
- 这是一个学习 Vue 3 的实践项目
- 现有代码库为刚初始化的 Vue 3 + TypeScript 模板
- 目标是通过完整的项目开发流程掌握 Vue 3 生态系统

**技术环境：**
- Vue 3.5.29 组合式 API
- TypeScript 5.9.3（严格模式）
- Vite 7.3.1 构建工具
- Pinia 状态管理
- Tailwind CSS v4（待集成）
- Vitest 测试框架

**开发模式：**
- TDD（测试驱动开发）- 核心逻辑 100% 测试覆盖
- Functional Core, Imperative Shell - 纯函数核心 + 命令式外壳
- 小步迭代，频繁提交

## 约束

- **技术栈**: Vue 3 + TypeScript + Vite + Tailwind CSS v4 — 现代标准技术栈，学习资源丰富
- **架构模式**: Functional Core, Imperative Shell — 核心逻辑纯函数，便于测试
- **测试覆盖**: 核心逻辑 100% — 保证代码质量和可维护性
- **性能**: 流畅 60fps 动画 — 良好的用户体验
- **兼容性**: 现代浏览器（Chrome/Firefox/Safari/Edge 最新两版本）
