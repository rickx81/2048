<task type="auto">
  <name>任务 2: 在 Tile.vue 实现动态 will-change 管理</name>
  <files>src/components/Tile.vue</files>
  <action>
    实现动态 will-change 管理，避免永久设置：

    1. **移除静态 will-change 类**：
       - 删除模板中的 `will-change-transform` 类
       - 移除任何静态的 will-change 样式

    2. **实现动态 isAnimating 状态**：
       ```typescript
       const isAnimating = ref(false)

       watch([() => props.value, isNew, isMerged], () => {
         if (isNew.value || isMerged.value) {
           // 动画开始前添加 will-change
           isAnimating.value = true
           // 动画结束后移除 will-change
           setTimeout(() => {
             isAnimating.value = false
           }, 200) // 与动画时长一致
         }
       })
       ```

    3. **条件应用 will-change 样式**：
       - 在模板中绑定 style 或 class
       - 仅在 isAnimating 为 true 时应用 will-change: transform, opacity

    4. **优化样式绑定**：
       - 确保 getTileStyle() 和 getTextStyle() 不使用触发布局的属性
       - 使用 transform 替代任何 left/top 定位

    参考研究文档中的"动态 will-change 管理"模式
  </action>
  <verify>
    <automated>grep -n "will-change" src/components/Tile.vue</automated>
  </verify>
  <done>Tile.vue 实现动态 will-change 管理，动画结束后自动移除</done>
