<task type="auto">
  <name>任务 4: 在 index.html 添加 FOUC 防止内联脚本</name>
  <files>index.html</files>
  <action>
    在 Vue 应用加载前读取 localStorage 并设置 data-theme 属性：

    **脚本位置：**
    在 <body> 标签开始后，<div id="app"> 之前添加内联脚本

    **脚本逻辑：**
    ```html
    <body>
      <script>
        (function() {
          try {
            const storedTheme = localStorage.getItem('2048-game-theme')
            if (storedTheme && ['neon', 'sky', 'forest', 'sunset', 'sakura'].includes(storedTheme)) {
              document.documentElement.dataset.theme = storedTheme
            }
          } catch (e) {
            console.warn('Failed to read theme from localStorage:', e)
            // 静默回退，使用默认主题
          }
        })()
      </script>
      <div id="app"></div>
      <script type="module" src="/src/main.ts"></script>
    </body>
    ```

    **要求：**
    - 使用 IIFE（立即执行函数）避免污染全局作用域
    - 验证存储的主题 ID 在有效范围内（5 个预设主题）
    - try-catch 捕获 localStorage 错误（隐私模式、配额超限等）
    - 失败时静默回退，不显示错误给用户
    - 仅在控制台记录警告

    **验证方法：**
    - 刷新页面时，主题立即应用（无闪烁）
    - localStorage 失败时使用默认主题

    **不允许：**
    - 不在 <head> 中添加脚本（避免阻塞渲染）
    - 不使用复杂的同步逻辑（保持简单）
  </action>
