# Phase 10 — UI Design Contract

> Visual and interaction contract for 音效系统 frontend phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |

## Spacing Scale

Declared values (基于现有组件推断的 4 的倍数系统):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | 图标内边距 |
| sm | 8px | 紧凑元素间距、控制按钮组间距 |
| md | 16px | 默认元素间距、页面内边距 |
| lg | 24px | 区域间距 |
| xl | 32px | 大区域内边距 |
| 2xl | 48px | 覆盖层内边距 |

**音量控制组件专用间距:**
- 音量按钮与现有控制按钮间距: 8px (0.5rem)
- 音量滑块容器内边距: 12px (0.75rem)
- 滑块与图标间距: 8px (0.5rem)

**移动端例外:**
- 音量按钮在移动端保持 44px 最小触摸目标（符合 WCAG 2.5.5）

## Color

使用 CSS 变量主题系统（5 个预设主题已定义）。

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | var(--theme-bg-primary) | 页面背景、主容器背景 |
| Secondary (30%) | var(--theme-bg-secondary) | 按钮、卡片、分数框、滑块轨道 |

**音量控制专用颜色:**
- 音量按钮背景: var(--theme-bg-secondary)
- 音量按钮悬停: opacity: 0.9

- 滑块轨道: var(--theme-bg-secondary)
- 滑块已填充轨道: var(--theme-text-primary)
- 滑块拇指: var(--theme-text-primary)
- 静音状态图标: var(--theme-text-secondary)

**Accent reserved for:** 无 — 项目使用多主题色彩系统，每个主题有独立的配色方案。

## 音量控制组件规范

### 组件名称
`VolumeControl.vue`

### 位置
`src/components/VolumeControl.vue`

### 视觉设计

**音量按钮（收起状态）:**
```
┌─────────────────────────┐
│  🔊  [撤销] [新游戏]     │
└─────────────────────────┘
```

**音量滑块（展开状态）:**
```
┌─────────────────────────────────┐
│  [🔊 ━━━━━━━━━●━━━ 75%]       │
└─────────────────────────────────┘
```

### 交互状态

| 状态 | 图标 | 行为 |
|------|------|------|
| 未静音 | 🔊 (speaker) | 点击展开滑块，拖动调节音量 |
| 已静音 | 🔇 (mute) | 滑块禁用，点击取消静音 |
| 悬停 | 放大 1.05 倍 | scale(1.05), opacity: 0.9 |
| 按下 | 缩小 0.95 倍 | scale(0.95), opacity: 0.8 |

### 滑块规格

- 轨道高度: 4px
- 轨道圆角: 2px
- 拇指大小: 16px × 16px
- 拇指圆角: 50%
- 滑块宽度: 120px
- 过渡动画: 0.15s ease

### 响应式行为

**桌面端 (> 640px):**
- 音量按钮在控制栏中显示
- 点击按钮展开/收起滑块

**移动端 (≤ 640px):**
- 音量按钮在控制栏中显示
- 点击按钮展开/收起滑块
- 滑块宽度自适应容器（min-width: 100px）

### 图标规范

使用内联 SVG，格式与现有按钮一致：
- 尺寸: 20px × 20px
- 描边宽度: 2
- 颜色: currentColor（继承文本颜色）

## 动画与过渡

| 交互 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮悬停 | 0.15s | ease |
| 按钮按下 | 0.10s | ease |
| 滑块展开 | 0.20s | ease-out |
| 滑块收起 | 0.15s | ease-in |
| 音量调节 | 0.05s | ease-out (实时响应) |

---

## 可访问性

- 所有按钮具有 `title` 属性描述功能
- 音量滑块具有 `aria-label="音量控制"`
- 音量滑块具有 `aria-valuemin="0"` 和 `aria-valuemax="100"`
- 静音状态通过 `aria-pressed` 传达
- 键盘导航: Tab 键聚焦，Enter/Space 触发

---

## 数据持久化

音量设置存储在 localStorage 中：
- 键名: `__GAME_2048_AUDIO_VOLUME__` (范围: 0-100)
- 静音状态: `__GAME_2048_AUDIO_MUTED__` (布尔值)
