<task type="auto">
  <name>任务 1: 重构 Tile.vue 使用 CSS 变量（方块颜色）</name>
  <files>src/components/Tile.vue</files>
  <action>
    将 Tile.vue 中的硬编码颜色映射替换为 CSS 变量引用：

    **getTileStyle 函数重构：**
    ```typescript
    function getTileStyle() {
      const baseStyle = {
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
      }

      if (props.value === 0) {
        return {
          ...baseStyle,
          backgroundColor: 'var(--theme-tile-empty)',
          borderRadius: '3px',
        }
      }

      return {
        ...baseStyle,
        backgroundColor: `var(--theme-tile-${props.value})`,
        borderRadius: '3px',
      }
    }
    ```

    **getTextStyle 函数重构：**
    ```typescript
    function getTextStyle() {
      if (props.value === 0) return {}

      const darkTextNumbers = [2, 4]
      const useDarkText = darkTextNumbers.includes(props.value)
      const fontSize = getFontSize(props.value)

      return {
        color: useDarkText ? 'var(--theme-text-secondary)' : 'var(--theme-text-primary)',
        fontSize,
        fontWeight: '700',
      }
    }
    ```

    **保持不变：**
    - getFontSize 函数（字体大小计算逻辑）
    - 动画状态管理（isNew, isMerged）
    - watch 监听器
    - CSS 类名（tile-new, tile-merged, will-change-transform）

    **移除：**
    - backgroundColors 常量映射对象（不再需要）
    - 硬编码颜色值（#cdc1b4, #eee4da, #f2b179 等）

    **验证点：**
    - 确保 props.value 在有效范围内（2-2048）
    - 如果 value 超出范围，CSS 变量会回退到默认值（浏览器原生行为）

    **不允许：**
    - 不迁移字体大小到 CSS 变量（保留 getFontSize 函数）
    - 不修改动画逻辑
    - 不改变组件结构或 props
  </action>

      # 验证 TypeScript 类型正确
    </automated>
  </verify>

    - Tile.vue 使用 var(--theme-text-*) 替代硬编码文字颜色
    - getFontSize 函数保持不变
