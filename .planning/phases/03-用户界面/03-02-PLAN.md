  <name>任务 2：创建游戏网格组件（GameBoard.vue）</name>

创建 `src/components/GameBoard.vue` 组件，渲染 4x4 游戏网格：

**脚本部分：**
```vue
&lt;script setup lang="ts"&gt;
import { computed } from 'vue'
import { useGameStore } from '@/stores/game'
import Tile from './Tile.vue'

const store = useGameStore()

// 将 4x4 网格展平为一维数组，便于 v-for 渲染
const tiles = computed(() =&gt; {
  const result: Array&lt;{ value: number; row: number; col: number; id: string }&gt; = []
  for (let row = 0; row &lt; 4; row++) {
    for (let col = 0; col &lt; 4; col++) {
      result.push({
        value: store.grid[row][col],
        row,
        col,
        id: `tile-${row}-${col}` // 唯一标识，用于 Vue key
      })
    }
  }
  return result
})
&lt;/script setup&gt;
```

**模板部分：**
```vue
&lt;template&gt;
  &lt;div class="game-board"&gt;
    &lt;div class="grid-container"&gt;
      &lt;Tile
        v-for="tile in tiles"
        :key="tile.id"
        :value="tile.value"
        class="grid-item"
      /&gt;
    &lt;/div&gt;
  &lt;/div&gt;
&lt;/template&gt;
```

**样式部分：**
```vue
&lt;style scoped&gt;
.game-board {
  width: 100%;
  aspect-ratio: 1; /* 保持正方形比例 */
}

.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(4, 1fr);
  gap: 0.75rem; /* 方格之间固定间距 */
  width: 100%;
  height: 100%;
  background-color: rgba(255, 255, 255, 0.1); /* 半透明玻璃态 */
  backdrop-filter: blur(10px);
  border-radius: 0.5rem;
  padding: 0.75rem;
  box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.3);
}

.grid-item {
  width: 100%;
  height: 100%;
}

/* 移动端适配 */
@media (max-width: 640px) {
  .grid-container {
    gap: 0.5rem;
    padding: 0.5rem;
  }
}
&lt;/style&gt;
```

**关键特性：**
- CSS Grid 布局（4x4 网格）
- 等宽等高单元格（repeat(4, 1fr)）
- 固定间距（gap: 0.75rem）
- 玻璃态背景（bg-white/10 + backdrop-blur）
- 正方形比例（aspect-ratio: 1）
- 响应式设计（移动端减小间距）
- 从 store 读取网格数据（store.grid）
- 使用唯一 key 标识每个方块（tile-row-col）

**网格背景说明：**
- 网格容器使用半透明背景
- 空位显示为透明（Tile 组件中 value === 0 时 bg-transparent）
- 方块带有阴影效果，与背景形成层次感
