  <name>移动和合并逻辑</name>

  <behavior>
    ## 基础行/列移动（单个维度）

    **向左移动一行：**
    - 输入: `[2, 0, 2, 4]` → 输出: `[4, 4, 0, 0]`, score: 4
    - 输入: `[2, 2, 2, 2]` → 输出: `[4, 4, 0, 0]`, score: 8
    - 输入: `[2, 4, 2, 4]` → 输出: `[2, 4, 2, 4]`, score: 0
    - 输入: `[4, 4, 4, 4]` → 输出: `[8, 8, 0, 0]`, score: 16

    ## 规则验证
    - 单次合并规则: `[2, 2, 4, 4]` → `[4, 8, 0, 0]`（不是 `[16, 0, 0, 0]`）
    - 滑动到底: `[0, 2, 0, 2]` → `[4, 0, 0, 0]`
    - 无合并时滑动: `[0, 0, 2, 4]` → `[2, 4, 0, 0]`
    - 全空: `[0, 0, 0, 0]` → `[0, 0, 0, 0]`, score: 0
    - 全满无合并: `[2, 4, 8, 16]` → `[2, 4, 8, 16]`, score: 0

    ## 四个方向移动

    **向左 (LEFT):**
    - 每行独立处理，每行向左移动和合并
    - 示例网格:
      ```
      [2, 2, 0, 4]    [4, 4, 0, 0]
      [0, 2, 2, 0] →  [4, 0, 0, 0]
      [4, 0, 4, 4]    [8, 4, 0, 0]
      [2, 4, 2, 4]    [2, 4, 2, 4]
      ```
      Score: 4 + 4 + 8 = 16

    **向右 (RIGHT):**
    - 每行独立处理，每行向右移动和合并
    - 示例网格（同上）:
      ```
      [2, 2, 0, 4]    [0, 0, 4, 4]
      [0, 2, 2, 0] →  [0, 0, 0, 4]
      [4, 0, 4, 4]    [0, 0, 4, 8]
      [2, 4, 2, 4]    [2, 4, 2, 4]
      ```
      Score: 4 + 4 + 8 = 16

    **向上 (UP):**
    - 每列独立处理，每列向上移动和合并
    - 示例网格:
      ```
      [2, 2, 0, 4]
      [0, 2, 2, 0]    [2, 4, 2, 4]
      [4, 0, 4, 4] →  [4, 2, 4, 8]
      [2, 4, 2, 4]    [2, 4, 4, 0]
      ```
      Score: 4 + 4 + 4 = 12

    **向下 (DOWN):**
    - 每列独立处理，每列向下移动和合并
    - 示例网格（同上）:
      ```
      [2, 2, 0, 4]
      [0, 2, 2, 0]    [0, 0, 2, 0]
      [4, 0, 4, 4] →  [2, 4, 4, 4]
      [2, 4, 2, 4]    [4, 2, 2, 8]
      ```
      Score: 4 + 4 + 4 = 12

    ## 有效移动判断

    **有效移动：** 网格发生变化（至少一个方块位置不同）
    - `[2, 0, 0, 0]` 向左 → 无变化，无效
    - `[2, 0, 0, 0]` 向右 → `[0, 0, 0, 2]`, 有效
    - 全满且无法合并 → 四个方向都无效

    ## 新方块生成（GAME-05）

    **有效移动后：**
    - 在随机空位生成一个新数字（2 或 4）
    - 90% 概率为 2，10% 概率为 4
    - 如果网格已满，不生成新数字

    **无效移动：**
    - 不生成新数字
    - 返回原网格和 score: 0

    ## 边界情况

    - 空网格移动后生成两个数字（初始化）
    - 只有一个空位时，必须在该位置生成
    - 全满但可以合并（如全是 2）→ 移动有效，生成新数字
    - 全满且无法合并 → 四个方向都无效，游戏结束

  ## 1. 辅助函数（单行/列处理）

  **slideRowLeft(row)**: 单行向左滑动并合并
  ```typescript
  function slideRowLeft(row: number[]): { newRow: number[]; score: number }
  ```
  步骤：
  1. 移除所有 0: `[2, 0, 2, 4]` → `[2, 2, 4]`
  2. 合并相邻相同数字: `[2, 2, 4]` → `[4, 4]`（score: 4）
     - 单次合并规则：每个数字只合并一次
     - 遍历数组，如果 `arr[i] === arr[i+1]`，合并为 `arr[i] * 2`，跳过 `i+1`
  3. 补齐 0 到长度 4: `[4, 4]` → `[4, 4, 0, 0]`

  **slideRowRight(row)**: 单行向右滑动并合并
  ```typescript
  function slideRowRight(row: number[]): { newRow: number[]; score: number }
  ```
  实现：反转行 → slideRowLeft → 反转结果

  **slideColumnUp(col)**: 单列向上滑动并合并
  ```typescript
  function slideColumnUp(col: number[]): { newCol: number[]; score: number }
  ```
  实现：复用 slideRowLeft 逻辑（列就是行）

  **slideColumnDown(col)**: 单列向下滑动并合并
  ```typescript
  function slideColumnDown(col: number[]): { newCol: number[]; score: number }
  ```
  实现：反转列 → slideColumnUp → 反转结果

  ## 2. 方向移动函数

  **moveLeft(grid)**: 向左移动
  ```typescript
  export function moveLeft(grid: Grid): { newGrid: Grid; score: number; moved: boolean }
  ```

  - 累加所有行的得分
  - 比较新旧网格，设置 moved 标志
  - 返回新网格（不可变操作）

  **moveRight(grid)**: 向右移动
  ```typescript
  export function moveRight(grid: Grid): { newGrid: Grid; score: number; moved: boolean }
  ```
  - 对每一行应用 slideRowRight
  - 其他逻辑同 moveLeft

  **moveUp(grid)**: 向上移动
  ```typescript
  export function moveUp(grid: Grid): { newGrid: Grid; score: number; moved: boolean }
  ```
  - 对每一列应用 slideColumnUp
  - 提取列 → 处理 → 放回网格
  - 其他逻辑同 moveLeft

  **moveDown(grid)**: 向下移动
  ```typescript
  export function moveDown(grid: Grid): { newGrid: Grid; score: number; moved: boolean }
  ```
  - 对每一列应用 slideColumnDown
  - 其他逻辑同 moveLeft

  ## 3. 主移动函数

  **move(grid, direction)**: 根据方向执行移动
  ```typescript
  export function move(grid: Grid, direction: Direction): MoveResult
  ```
  其中 MoveResult 定义为：
  ```typescript
  interface MoveResult {
    grid: Grid;           // 移动后的网格
    score: number;        // 本次移动得分
    moved: boolean;       // 是否发生了移动
  }
  ```

  实现：
  - 根据 direction 参数调用对应的 move* 函数
  - 如果 moved 为 true 且有空位，调用 addRandomTile 生成新数字
  - 返回最终结果

  ## 4. 辅助工具函数

  **extractColumn(grid, colIndex)**: 提取指定列
  ```typescript
  function extractColumn(grid: Grid, colIndex: number): number[]
  ```

  **insertColumn(grid, colIndex, column)**: 插入列到指定位置
  ```typescript
  function insertColumn(grid: Grid, colIndex: number, column: number[]): Grid
  ```

  **gridsEqual(grid1, grid2)**: 比较两个网格是否相同
  ```typescript
  function gridsEqual(grid1: Grid, grid2: Grid): boolean
  ```

  ## 实现要点

  - 所有函数都是纯函数，不修改输入参数
  - 使用深拷贝创建新网格
  - 单次合并规则：每个数字每次移动最多合并一次
  - 正确累加分数
  - 只在有效移动后生成新数字
  - 代码复用：slideRow* 和 slideColumn* 共享逻辑
